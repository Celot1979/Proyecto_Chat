# Proyecto_Chat
Este proyecto es la continuación del curso de Java que cursé. Donde la principal función era realizar un chat ( LAN ) para afianzar conocimientos adquiridos.

- Swing.
- Threads.
- ArrayList
- String de datos
- Sockets
